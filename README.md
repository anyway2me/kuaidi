# kuaidi
A command tool for querying express delivery(Based on [kuaidi100](http://www.kuaidi100.com/))

![](http://i1.tietuku.com/8fb526d5a099bdf2.png)

## Install

```bash
$ npm install kuaidi -g
```

## Usage

```bash
$ kuaidi 200093247451
```
